# Assignment: Simple Algorithms - GCD Calculator

## Problem Statement
Write a Cairo program that computes the Greatest Common Divisor (GCD) of two integers using the Euclidean algorithm and prints the result.

## Requirements Covered
- Implemented in Cairo language.
- Includes comments in code.
- Includes detailed topic explanation in guide.md.
- Structured to run with Scarb.

## Files
- `src/lib.cairo` - executable Cairo implementation.
- `gcd_calculator.cairo` - starter/standalone assignment file.
- `guide.md` - detailed explanation of topic and implementation.
- `Scarb.toml` - Scarb package configuration.

## Sample Output
```text
First number: 48
Second number: 18
GCD(48, 18) = 6
```

## Run
From this folder:

```bash
scarb execute
```
